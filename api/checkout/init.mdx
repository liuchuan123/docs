---
title: "初始化支付"
description: "POST /api/web/checkout/init 初始化收银台支付"
openapi: "POST /api/web/checkout/init"
---


用户选择支付方式后，调用此接口初始化支付，获取渠道信息和渲染所需参数。

## 接口信息

| 项目 | 说明 |
| --- | --- |
| 请求方式 | `POST` |
| 路径 | `/api/web/checkout/init` |
| Content-Type | `application/json` |
| 鉴权方式 | 收银台会话签名 |
| 端口 | `10030` |

## 请求参数

```json
{
  "orderId": "1000126011700000100001",
  "paymentMethodCode": "card"
}
```

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `orderId` | String | 是 | 订单 ID |
| `paymentMethodCode` | String | 是 | 支付方式编码（来自支付方式列表接口） |

## 响应参数

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "channelType": "Stripe",
    "transactionId": "TXN20260118001",
    "paymentUrl": null,
    "cardBrands": [
      { "key": "visa", "name": "Visa" },
      { "key": "mastercard", "name": "Mastercard" }
    ],
    "extension": {
      "paymentRenderType": "STRIPE_ELEMENT",
      "clientSecret": "pi_xxx_secret_xxx"
    }
  }
}
```

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `channelType` | String | 渠道类型，如 `Stripe`、`Antom`、`Sandbox` |
| `transactionId` | String | 交易 ID |
| `paymentUrl` | String | 重定向支付地址（`REDIRECT` 模式时非空） |
| `cardBrands` | Array | 支持的卡品牌列表 |
| `cardBrands[].key` | String | 卡品牌编码 |
| `cardBrands[].name` | String | 卡品牌名称 |
| `extension.paymentRenderType` | String | 渲染方式，见下方枚举 |

### 渲染方式枚举

| 值 | 说明 | 前端处理 |
| --- | --- | --- |
| `STRIPE_ELEMENT` | 使用 Stripe Elements 渲染 | 用 `clientSecret` 初始化 Stripe.js |
| `COMPONENT_SDK` | 使用渠道自有 SDK 渲染 | 加载渠道 SDK 并挂载组件 |
| `OWN_FORM` | 商户自建表单 | 自行采集卡信息提交 |
| `REDIRECT` | 重定向支付 | 将用户跳转到 `paymentUrl` |

## 请求示例

```bash
curl -X POST "https://cashier-api.paymatrixpay.com/api/web/checkout/init" \
  -H "Content-Type: application/json" \
  -H "X-Session-Id: <SESSION_ID>" \
  -d '{
    "orderId": "1000126011700000100001",
    "paymentMethodCode": "card"
  }'
```

## 下一步

根据返回的 `paymentRenderType` 进行对应的前端渲染，然后调用 [提交支付](/api/checkout/submit)。
