---
title: "获取支付方式"
description: "GET /api/web/checkout/payment-methods 获取订单可用的支付方式列表"
openapi: "GET /api/web/checkout/payment-methods"
---


根据订单 ID 获取该订单支持的支付方式列表，前端据此渲染支付方式选择器。

## 接口信息

| 项目 | 说明 |
| --- | --- |
| 请求方式 | `GET` |
| 路径 | `/api/web/checkout/payment-methods` |
| 鉴权方式 | 收银台会话签名 |
| 端口 | `10030` |

## 请求参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `orderId` | String | 是 | 订单 ID（URL Query 参数） |

## 响应参数

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "payment_methods": [
      {
        "code": "card",
        "name": "银行卡",
        "icon": "https://cdn.paymatrixpay.com/icons/card.svg"
      }
    ]
  }
}
```

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `payment_methods` | Array | 可用支付方式列表 |
| `payment_methods[].code` | String | 支付方式编码，如 `card`、`alipay` |
| `payment_methods[].name` | String | 支付方式名称 |
| `payment_methods[].icon` | String | 支付方式图标 URL |

## 请求示例

```bash
curl -X GET "https://cashier-api.paymatrixpay.com/api/web/checkout/payment-methods?orderId=1000126011700000100001" \
  -H "X-Session-Id: <SESSION_ID>"
```

## 下一步

获取到支付方式后，调用 [初始化支付](/api/checkout/init) 获取支付参数。
