---
title: "提交支付"
description: "POST /api/web/payment/submit 提交支付请求"
openapi: "POST /api/web/payment/submit"
---


用户填写完支付信息后，调用此接口提交支付。支持银行卡支付等场景。

## 接口信息

| 项目 | 说明 |
| --- | --- |
| 请求方式 | `POST` |
| 路径 | `/api/web/payment/submit` |
| Content-Type | `application/json` |
| 鉴权方式 | 收银台会话签名 |
| 端口 | `10030` |

## 请求参数

```json
{
  "orderId": "1000126011700000100001",
  "terminal": {
    "userClientBrowserColorDepth": "32",
    "userClientBrowserScreenHeight": "1080",
    "userClientBrowserScreenWidth": "1920",
    "userClientBrowserTimeZoneOffset": "-480"
  },
  "card": {
    "holderName": "Zhang San",
    "cardNo": "4242424242424242",
    "cardExpirationYear": "28",
    "cardExpirationMonth": "12",
    "cardBrand": "Visa",
    "cvv": "123",
    "billingAddress": {
      "country": "US",
      "state": "CA",
      "city": "San Francisco",
      "street": "123 Main St",
      "zip": "94105"
    },
    "email": "customer@example.com"
  },
  "deviceId": "device_fingerprint_abc123"
}
```

### 终端信息（terminal）

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `userClientBrowserColorDepth` | String | 是 | 浏览器颜色深度，如 `32` |
| `userClientBrowserScreenHeight` | String | 是 | 浏览器屏幕高度，如 `1080` |
| `userClientBrowserScreenWidth` | String | 是 | 浏览器屏幕宽度，如 `1920` |
| `userClientBrowserTimeZoneOffset` | String | 是 | UTC 时区偏移（分钟），如北京时间填 `-480` |

### 卡信息（card）

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `holderName` | String | 是 | 持卡人姓名，最长 128 字符 |
| `cardNo` | String | 是 | 银行卡号，12-19 位 |
| `cardExpirationYear` | String | 是 | 有效期年份，2 位，如 `28` |
| `cardExpirationMonth` | String | 是 | 有效期月份，2 位，如 `12` |
| `cardBrand` | String | 是 | 卡品牌：`Visa`、`MasterCard`、`Amex`、`JCB`、`Discover`、`DinersClub`、`UnionPay` |
| `cvv` | String | 是 | 安全码，3-4 位数字 |
| `billingAddress` | Object | 是 | 账单地址 |
| `billingAddress.country` | String | 是 | 国家代码 |
| `billingAddress.state` | String | 否 | 州/省 |
| `billingAddress.city` | String | 是 | 城市 |
| `billingAddress.street` | String | 是 | 街道地址 |
| `billingAddress.zip` | String | 是 | 邮政编码 |
| `email` | String | 是 | 邮箱地址 |

### 设备信息

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `deviceId` | String | 否 | 设备指纹 ID，来自 `X-Device-Id` 请求头 |

## 响应参数

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "transactionId": "TXN20260118001",
    "paymentUrl": null
  }
}
```

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `transactionId` | String | 交易 ID |
| `paymentUrl` | String | 3DS 认证重定向地址，非空时需要将用户跳转到此 URL |

## 请求示例

```bash
curl -X POST "https://cashier-api.paymatrixpay.com/api/web/payment/submit" \
  -H "Content-Type: application/json" \
  -H "X-Session-Id: <SESSION_ID>" \
  -H "X-Device-Id: device_fingerprint_abc123" \
  -d '{
    "orderId": "1000126011700000100001",
    "terminal": {
      "userClientBrowserColorDepth": "32",
      "userClientBrowserScreenHeight": "1080",
      "userClientBrowserScreenWidth": "1920",
      "userClientBrowserTimeZoneOffset": "-480"
    },
    "card": {
      "holderName": "Zhang San",
      "cardNo": "4242424242424242",
      "cardExpirationYear": "28",
      "cardExpirationMonth": "12",
      "cardBrand": "Visa",
      "cvv": "123",
      "billingAddress": {
        "country": "US",
        "state": "CA",
        "city": "San Francisco",
        "street": "123 Main St",
        "zip": "94105"
      },

```

## 支付结果处理

<Warning>
  提交支付成功后，前端可能会收到 `paymentUrl`（3DS 重定向地址）。最终支付结果须以 [Webhook 通知](/api/webhook/overview) 为准，不可仅信任前端返回。
</Warning>
