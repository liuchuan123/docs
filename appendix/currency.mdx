# 支持币种

## 交易币种

以下为当前支持的支付交易币种：

| 币种代码 | 币种名称       | 最小单位 |
| -------- | -------------- | -------- |
| `USD`    | 美元           | 0.01     |
| `EUR`    | 欧元           | 0.01     |
| `GBP`    | 英镑           | 0.01     |
| `CNY`    | 人民币         | 0.01     |
| `JPY`    | 日元           | 1        |
| `HKD`    | 港币           | 0.01     |
| `SGD`    | 新加坡元       | 0.01     |
| `AUD`    | 澳大利亚元     | 0.01     |
| `CAD`    | 加拿大元       | 0.01     |

> 日元（JPY）的最小单位是 1，无小数位。

## 结算币种

结算币种指资金打入商户银行账户时使用的币种，默认为交易币种。

目前支持以下结算币种：

- `USD` — 美元
- `EUR` — 欧元
- `CNY` — 人民币
- `HKD` — 港币

## 币种转换

如果交易币种与结算币种不同，平台在结算时会自动进行币种转换：

- 汇率以结算日当天银行间汇率为准
- 转换费用在手续费明细中体现

## 金额格式注意事项

- 所有金额使用 `BigDecimal` 类型，JSON 中为数值类型
- 最多两位小数（日元除外）
- 示例：`100.00` 表示 100 美元
- 不要使用科学计数法：错误 `1.0E2`，正确 `100.00`
- 不要使用千分位分隔符：错误 `1,000.00`，正确 `1000.00`
