商户入驻是接入 PayMatrix 的第一步。完成入驻后，你将获得商户 ID、API 调用权限以及商户门户的完整功能。

## 入驻条件

### 企业资质要求

- 持有合法有效的企业营业执照
- 企业法人身份信息真实有效
- 有正常经营的网站或应用（用于业务审核）

### 行业限制

以下行业暂不支持入驻：

- 虚拟货币、加密货币交易
- 成人内容、赌博
- 武器、毒品等违禁品
- 其他法律法规禁止的行业

## 入驻流程

<Steps>
  <Step title="注册账号">
    在商户门户使用邮箱注册账号，完成邮箱验证。
  </Step>
  <Step title="提交企业资料">
    填写企业基本信息，上传营业执照、法人身份证等资质文件。
  </Step>
  <Step title="填写企业资料">
    填写企业基本信息、法人信息，上传营业执照等资质文件。
  </Step>
  <Step title="平台审核">
    平台将在 1-3 个工作日内完成资质审核，审核结果将通过邮件和短信通知。
  </Step>
  <Step title="签约开通">
    审核通过后，在线签署服务协议，开通商户服务和 API 权限。
  </Step>
</Steps>

## 入驻后可以做什么

入驻审核通过后，即可获得：

- **商户门户**使用权：查看交易数据、管理订单、处理退款、查看结算
- **沙盒环境**权限：立即开始 API 联调测试
- **生产环境**权限：签约后自动开通
