# 注册流程

## 第一步：创建账号

1. 访问商户门户注册页面
2. 输入邮箱地址，设置登录密码
3. 点击发送验证码，输入邮箱收到的验证码
4. 完成邮箱验证后，账号创建成功

## 第二步：填写企业信息

登录商户门户后，进入「账户中心 → 商户信息」填写以下内容：

### 基本信息

| 字段       | 说明                       | 要求               |
| ---------- | -------------------------- | ------------------ |
| 企业名称   | 营业执照上的完整名称       | 必填，2-100 字符   |
| 统一社会信用代码 | 营业执照上的统一社会信用代码 | 必填，18 位       |
| 企业类型   | 企业 / 个体工商户          | 必填               |
| 注册地址   | 营业执照上的注册地址       | 必填               |
| 经营范围   | 主营业务描述               | 必填，最多 500 字  |
| 企业网站   | 公司官网或应用链接         | 选填               |

### 法人信息

| 字段         | 说明         | 要求     |
| ------------ | ------------ | -------- |
| 法人姓名     | 法人真实姓名 | 必填     |
| 法人身份证号 | 法人身份证号 | 必填     |
| 法人手机号   | 法人联系手机 | 必填     |

## 第三步：上传资质文件

需要上传的资质文件：

| 文件          | 要求                       |
| ------------- | -------------------------- |
| 营业执照      | 彩色扫描件或清晰照片，四角完整可见 |
| 法人身份证正面 | 彩色扫描件或清晰照片       |
| 法人身份证反面 | 彩色扫描件或清晰照片       |
| 银行开户许可证 | 或提供银行账户信息截图     |

文件格式要求：
- 支持 JPG、PNG、PDF 格式
- 单文件不超过 5MB
- 需清晰可辨，无遮挡、反光

## 第四步：填写结算银行账户

| 字段       | 说明         |
| ---------- | ------------ |
| 开户银行   | 银行全称     |
| 开户支行   | 支行名称     |
| 银行账号   | 对公银行账号 |
| 开户名称   | 需与营业执照名称一致 |

## 第五步：提交审核

确认信息无误后，点击提交。平台将在 1-3 个工作日内完成审核。

<Tip>
  建议在提交前仔细核对所有信息，信息不一致会导致审核被驳回，延长入驻时间。
</Tip>

## 审核状态说明

| 状态     | 说明                         |
| -------- | ---------------------------- |
| 待提交   | 尚未完成资料填写             |
| 审核中   | 已提交，等待平台审核         |
| 审核通过 | 资质审核通过，可签约开通服务 |
| 已驳回   | 审核未通过，需按驳回原因修改后重新提交 |
