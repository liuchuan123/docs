## 审核内容

平台将从以下几个方面进行审核：

### 企业真实性

- 营业执照是否真实有效（通过国家企业信用信息公示系统核验）
- 企业名称、统一社会信用代码是否一致
- 法人信息与营业执照是否匹配

### 业务合规性

- 经营范围是否符合平台准入要求
- 网站/应用内容是否合规
- 不存在违规行业

### 资质文件质量

- 文件是否清晰可辨
- 文件是否在有效期内
- 是否提供了完整的文件清单

## 审核周期

- 正常审核：1-3 个工作日
- 如遇节假日或资料不完整，可能延长

## 常见驳回原因及处理

| 驳回原因                   | 处理方法                     |
| -------------------------- | ---------------------------- |
| 营业执照模糊不清           | 重新拍摄/扫描清晰的营业执照  |
| 法人姓名与营业执照不一致   | 核实信息后重新提交           |
| 经营范围涉及禁止行业       | 确认业务范围，如有误可申诉   |
| 银行开户名称与营业执照不符 | 提供正确的对公账户信息       |
| 网站无法访问或无内容       | 确保网站正常可访问           |

## 审核结果通知

审核结果将通过以下方式通知：

- **邮件通知**：发送至注册邮箱
- **短信通知**：发送至法人手机号（如已配置）
- **门户通知**：登录商户门户可查看审核状态

## 审核驳回后的处理

1. 登录商户门户查看驳回原因
2. 按提示修改对应信息
3. 重新提交审核

驳回后重新提交不限制次数。
