---
title: "账户管理"
description: "管理商户账户信息、修改密码、安全设置"
---


「账户管理」模块用于管理商户的企业信息、登录凭据和账户安全。

## 企业信息

查看和修改企业基本信息：

- **企业名称**：营业执照上的企业名称
- **统一社会信用代码**：企业的唯一标识
- **企业类型**：企业 / 个体工商户
- **注册地址**：企业注册地址
- **经营范围**：主营业务描述

企业名称和统一社会信用代码等关键信息修改后需要平台重新审核。

## 法人信息

查看和修改法人代表信息：
- 姓名
- 身份证号
- 联系电话

## 结算银行账户

管理用于资金结算的银行账户：

- 支持绑定多个银行账户
- 需设置一个默认结算账户
- 修改银行账户需要重新审核

### 银行账户信息

| 字段     | 说明                     |
| -------- | ------------------------ |
| 开户银行 | 银行全称                 |
| 开户支行 | 支行名称（选填）         |
| 银行账号 | 对公银行账号             |
| 开户名称 | 需与营业执照上的名称一致 |

## 登录安全

### 修改密码

1. 输入当前密码
2. 输入新密码（需满足复杂度要求）
3. 确认新密码
4. 提交

### 多因素认证（MFA）

支持两种 MFA 方式：

- **TOTP 动态码**：使用 Google Authenticator 或类似 App 绑定
- **邮箱验证码**：登录时发送至注册邮箱

建议开启 MFA 以增强账户安全。

### 登录历史

查看账户的登录记录：
- 登录时间
- 登录 IP
- 登录设备
- 是否成功

如发现异常登录，请立即修改密码并联系平台。

## 子账户管理（如有）

部分商户可能需要创建子账户，用于团队协作：

- 支持创建多个子账户
- 可为子账户分配不同角色权限
- 子账户的操作日志独立可查
