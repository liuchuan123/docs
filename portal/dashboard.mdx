## 核心指标卡片

Dashboard 顶部以卡片形式展示当日核心指标：

| 指标 | 说明 |
| ---- | ---- |
| 今日交易笔数 | 当日成功支付的交易数量 |
| 今日交易金额 | 当日成功支付的交易总额 |
| 成功率 | 当日支付成功率 |
| 待结算 | 当前尚未结算的总金额 |

## 交易趋势图

Dashboard 以图表展示近期的交易趋势，便于你观察业务波动，及时发现异常。

## 最近交易

展示最近的交易记录，包含交易时间、订单 ID、交易金额、支付方式与交易状态。点击任一条记录可在 [交易管理](/portal/orders#交易详情) 中查看交易详情。

## 账户余额

Dashboard 展示商户账户的四种余额：

| 余额类型 | 说明 |
| -------- | ---- |
| 待结算 | 已成功交易但尚未到达结算周期的金额（**已扣除手续费与保证金计提**，非交易总额） |
| 可提现 | 已完成结算、可申请提现的可用余额 |
| 冻结 | 因争议等原因暂时冻结的金额 |
| 保证金 | 预留的风险保证金 |

<Tip>
  手续费与保证金在交易入账时即已扣除，因此「待结算」小于交易总额。余额明细与资金流转说明可在 [结算管理](/portal/settlements) 中进一步查看，结算周期与规则见 [结算体系](/concepts/settlement)。
</Tip>
