商户门户 Dashboard 是登录后的首页，提供核心业务的概览视图。

## 核心数据卡片

Dashboard 顶部展示当日核心指标：

| 指标           | 说明                       |
| -------------- | -------------------------- |
| 今日交易笔数   | 当日成功支付的交易数量     |
| 今日交易金额   | 当日成功支付的交易总额     |
| 成功率         | 当日支付成功率             |
| 待结算金额     | 当前尚未结算的总金额       |

## 交易趋势图

Dashboard 提供近 7 日 / 30 日的交易趋势图表：
- 交易金额趋势（折线图）
- 交易笔数趋势（柱状图）
- 支付方式分布（饼图）

## 最近交易列表

展示最近的交易记录，包含：
- 交易时间
- 订单 ID
- 交易金额
- 支付方式
- 交易状态

点击可进入交易详情页面。

## 账户余额

展示各账户余额：

| 账户类型 | 说明                 |
| -------- | -------------------- |
| 可用余额 | 可随时提现的余额     |
| 冻结余额 | 因争议等原因冻结的余额 |
| 保证金   | 预留的风险保证金     |
| 待结算   | 已交易但尚未结算的金额 |

## 快捷入口

- **创建测试订单**：快速在沙盒环境创建测试订单
- **查看结算单**：查看最新结算单和预计到账时间
- **开发者设置**：管理 API 密钥和 Webhook 配置
- **文档中心**：跳转到本文档
