## 支付订单列表

按时间范围、订单状态、支付方式等条件筛选查看：

| 字段         | 说明                               |
| ------------ | ---------------------------------- |
| 订单 ID      | 平台生成的唯一订单号               |
| 商户订单号   | 商户创建支付时传入的交易 ID        |
| 金额         | 订单金额与币种                     |
| 支付方式     | 用户使用的支付方式                 |
| 状态         | 订单当前状态                       |
| 创建时间     | 订单创建时间                       |
| 支付成功时间 | 用户实际完成支付的时间             |

支持导出为 Excel 文件。

## 交易详情

点击订单可以查看详情，包含：

- 订单基本信息（ID、金额、币种、状态、时间）
- 商品信息（商品列表、单价、数量）
- 客户信息（姓名、地址）
- 收货信息
- 支付流水（每次支付尝试的详情）
- 退款记录（如有关联退款）
- 费用明细（平台手续费、渠道手续费）

## 退款管理

退款列表展示所有退款记录：

| 字段             | 说明             |
| ---------------- | ---------------- |
| 退款 ID          | 平台退款单号     |
| 商户退款号       | 商户传入的退款号 |
| 原订单 ID        | 关联的支付订单   |
| 退款金额         | 退款金额与币种   |
| 状态             | 退款处理状态     |
| 退款时间         | 退款发起时间     |
| 退款原因         | 商户填写的退款原因 |

### 发起退款

在交易详情页面点击「退款」按钮：

1. 输入退款金额（支持部分退款，不超过可退金额）
2. 填写退款原因和备注
3. 确认提交

退款结果将通过 Webhook 通知。

## 争议管理

争议（Dispute / Chargeback）列表展示所有争议订单，包含争议状态、争议金额、应诉截止时间等信息。

点击争议可查看详情和提交抗辩材料。

## 交易报表

支持按日/周/月生成交易汇总报表：
- 交易笔数、金额汇总
- 渠道分布
- 退款汇总
- 手续费汇总

可导出为 CSV / Excel 格式用于对账。
