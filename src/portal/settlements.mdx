「结算管理」模块用于查看资金结算情况、结算明细，以及申请提现。

<Info>
  结算周期和费率以签约协议为准。如有疑问请联系客户经理或查看 [结算体系说明](/concepts/settlement)。
</Info>

## 结算单列表

展示所有结算批次：

| 字段 | 说明 |
| ---- | ---- |
| 结算批次号 | 平台生成的结算批次唯一编号 |
| 结算周期 | 覆盖的交易时间范围 |
| 结算金额 | 本期结算总金额（结算币种） |
| 手续费 | 本期平台手续费总额 |
| 状态 | 结算处理状态 |
| 到账时间 | 资金到商户账户余额的时间 |

列表支持导出为 Excel 文件，用于财务对账。

<Note>
  平台没有独立的「报表中心」，各类导出均按当前列表逐项生成 Excel。你可在结算单列表、[交易列表](/portal/orders) 等页面分别导出对账数据。
</Note>

## 结算明细

点击结算批次可查看明细：

- 每笔交易的结算金额
- 每笔交易的手续费
- 币种转换明细（如有）
- 商户账户信息

## 提现

结算资金到达可提现（可用）余额后，你可在本模块申请提现。

<Steps>
  <Step title="发起提现申请">
    选择提现金额，发起提现申请。
  </Step>
  <Step title="填写收款银行账户">
    提现时填写收款银行账户信息：户名、账号、银行名称。**无需预先绑定银行卡**。
  </Step>
  <Step title="提交并等待到账">
    确认后提交，等待资金到达你的银行账户。
  </Step>
</Steps>

<Tip>
  提现的收款银行账户在每次提现时填写，平台不要求你预先绑定或保存银行卡信息。
</Tip>

## 账户余额说明

商户账户共有四种余额：

| 余额类型 | 说明 |
| -------- | ---- |
| 待结算 | 已成功交易但尚未到达结算周期的金额，将按周期结转为可提现余额 |
| 可提现 | 已完成结算、可随时申请提现的可用余额 |
| 冻结 | 因争议等原因暂时冻结的金额，处理结束后解冻 |
| 保证金 | 预留的风险保证金 |

交易完成后的资金流转路径：

```
支付成功 → 待结算 →（到达结算日）→ 可提现余额 →（提现）→ 银行账户
```
